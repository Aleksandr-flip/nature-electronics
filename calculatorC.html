Калькулятор подбора конденсаторов

CТЭ(nF)

CП(nF)

В ситуации необходимости увеличения емкости конденсатора для временной проверки, без выпаивания его из печатной платы настраиваемого устройства, до требуемого эквивалентного значения, можно подключить дополнительный конденсатор. Для вычисления емкости добавочного конденсатора, подключаемого параллельно конденсатору находящемуся на плате устройства можно воспользоваться этим калькулятором.

CТЭ – требуемая эквивалентная емкость двух параллельных конденсаторов,

CП – емкость конденсатора находящегося на плате настраиваемого устройства к которому будет подключен дополнительный конденсатор,

CД – емкость дополнительного, внешнего конденсатора необходимого для получения требуемой, эквивалентной емкости.
